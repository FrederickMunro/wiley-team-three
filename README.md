### Getting Started
1. Install [Node.js](https://nodejs.org/en/download).
2. Navigate to the frontend directory:
   ```bash
   cd wiley-team-three/frontend
   ```
3. Install node project dependencies:
   ```bash
   npm install
   ```
4. Run the development environment:
   ```bash
   npm run dev
   ```
